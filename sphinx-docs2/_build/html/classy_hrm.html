classy_hrm module¶

classy_hrm.classy_hrm(file='ecg_data1.csv', window=20, brady_max=40, tachy_min=100)¶
Creates ECG class given input parameters and performs analysis
on data in file to find heart condition, as well as instantaneous and average heart rate in BPM. These results are printed to <file>_out.txt and are printed to the console.
Parameters:
  • file – str, name of input csv file
  • window – int, window size in seconds to divide signal into for diagnosis
  • brady_max – int, upper threshold for bradycardia in BPM
  • tachy_min – int, lower threshold for tachycardia in BPM
Returns:

none
